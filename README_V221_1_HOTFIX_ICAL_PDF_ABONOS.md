# V221.1 Hotfix iCal / PDF / Abonos congelados

Cambios quirúrgicos sobre V221:

1. El botón **Sincronizar iCal guardado** queda protegido contra doble ejecución y nunca desvincula URLs ni calendarios guardados. Si un iCal falla, conserva bloqueos previos y muestra error.
2. Se elimina el texto residual **Documento / Formato genérico** de PDFs genéricos.
3. El bloque **Abono equivalente $** y **Diferencia a pagar** usa importes congelados del historial de abonos, evitando recalcular abonos antiguos con la tasa del día.

No se modifica caja, reservas, Renta Car, alojamientos, aliados, iCal export, Firebase, inventario, RRHH ni ROI.
