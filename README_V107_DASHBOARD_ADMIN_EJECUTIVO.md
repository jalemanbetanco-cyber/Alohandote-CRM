# V107 - Dashboard Administrativo Ejecutivo

## Objetivo
Mejorar la lectura rápida del módulo Administración ERP reduciendo la sensación de gris y creando una jerarquía visual más clara.

## Cambios UX/UI

### 1. Resumen ejecutivo
Se crea una tarjeta principal:
- Caja disponible
- USD principal
- Bs secundario

### 2. Tarjetas por categoría
Se separan los indicadores principales:
- Ingresos
- Egresos
- Utilidad estimada
- Por cobrar

Cada uno tiene fondo suave por categoría:
- Ingresos: verde claro
- Egresos: rojo claro
- Utilidad: azul claro
- Por cobrar: amarillo claro

### 3. Alertas administrativas
Se agrupan:
- Por pagar
- Devoluciones
- Comisiones

Con fondo suave y jerarquía de alerta.

### 4. Caja por método de pago
Nueva sección visual:
- Efectivo $
- Zelle
- Binance
- Bs
- Sin método

Incluye íconos y tarjetas separadas para lectura rápida.

### 5. Responsive
En mobile las tarjetas se reorganizan en una columna para lectura vertical clara.

## No cambia
- Cálculos
- Compra de $
- Exportación administración
- Tablas administrativas
- Reglas de caja
