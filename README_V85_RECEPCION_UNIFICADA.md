# V85 - Recepción unificada

## Cambios
- El módulo deja de llamarse "Recepción vehículos" y ahora se llama **Recepción**.
- El dashboard de recepción ahora incluye:
  - Vehículos por recibir según fecha de entrega.
  - Alojamientos por recibir/revisar según fecha de salida/check-out.
- Pendientes de recepción hoy incluye:
  - Vehículos con entrega hoy o vencidos.
  - Alojamientos con salida/check-out hoy o vencidos.
- Próximas recepciones solo muestra eventos dentro de los próximos 2 días.
- Las recepciones futuras posteriores a 2 días no se muestran para no saturar el dashboard.

## No se modifica
- Renta Car
- Alojamientos
- Cotizaciones
- Reservas
- Administración ERP
- Inventario ERP
- RRHH ERP
- Mantenimiento
- Rentabilidad KM / ROI
- iCal múltiple
