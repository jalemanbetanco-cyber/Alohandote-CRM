# V81 Fix - Administración ERP visible

## Corrección
La V81 anterior tenía el módulo Administración, pero podía no verse claramente dentro de la navegación.

## Cambios
- El botón ahora se llama **Administración ERP**.
- El botón queda visible en la barra superior, sin depender del perfil.
- Se agrega un banner de acceso rápido: **Abrir Administración ERP**.
- Se conserva el dashboard administrativo con:
  - Ingresos
  - Egresos
  - Utilidad estimada
  - Cuentas por cobrar
  - Cuentas por pagar
  - Devoluciones
  - Caja general
  - Comisiones
- No se modifica la lógica de Renta Car, Alojamientos, Recepción, Mantenimiento, ROI ni iCal múltiple.
