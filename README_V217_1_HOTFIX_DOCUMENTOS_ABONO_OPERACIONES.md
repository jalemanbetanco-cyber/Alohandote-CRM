# V217.1 Hotfix Documentos / Abono Obligatorio / Operaciones

Alcance quirúrgico sobre V217 estable.

## Cambios
- Elimina la línea naranja bajo el título de recibos/cotizaciones PDF.
- Agrega emojis al footer de contacto PDF: Instagram, WhatsApp y correo.
- Si el método de pago es distinto a Bs, cotizaciones y recibos muestran solo montos en USD.
- Si el método de pago es Bs, mantiene USD / Bs y tasa EURO como referencia.
- Bloquea creación/guardado de reservas sin abono mayor a 0 con mensaje: "Ingresar abono para reservar.".
- Links públicos de operaciones pasan a vigencia de 30 días para facilitar uso por colaboradores externos.
