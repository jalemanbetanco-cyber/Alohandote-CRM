# V198 · Volver al formulario desde documentos

## Alcance puntual

Se corrige únicamente el comportamiento del botón **Volver a la app** en cotizaciones, recibos, contratos y comprobantes imprimibles.

## Regla aplicada

- El documento se abre directamente en una pestaña hija.
- La aplicación principal permanece abierta con el formulario y sus datos intactos.
- Al presionar **Volver a la app**, se enfoca la pestaña principal y se cierra la pestaña del documento.
- Si el navegador bloquea la pestaña, se conserva el mecanismo anterior como respaldo.

## Fuera de alcance / no modificado

- iCal Airbnb y Estei.
- Cajas y movimientos financieros.
- Gastos, cuentas por pagar y ROI.
- Kilometraje, entregas y recepciones.
- Tasas BCV.
- Reservas, mantenimiento, RRHH, inventario y reglas Firebase.
