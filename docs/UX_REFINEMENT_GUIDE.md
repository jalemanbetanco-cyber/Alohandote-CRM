# V131 - Guía de optimización UX mobile/web

## Objetivo
Mejorar usabilidad sin alterar la lógica operativa del sistema.

## Mejoras aplicadas
- Botones táctiles mínimos.
- Inputs a 16px para evitar zoom accidental en iPhone.
- Focus visible para teclado.
- Tablas con scroll horizontal controlado.
- Modales con altura segura.
- Acciones principales más cómodas en mobile.
- Respeto a `prefers-reduced-motion`.
- Limpieza visual para impresión.

## Zonas impactadas
- Administración ERP.
- Mantenimiento.
- Rentabilidad.
- Formularios.
- Tablas.
- Login / link público.
- Modales.

## Regla
No se cambió lógica de negocio.
Solo se aplicaron mejoras de presentación, accesibilidad y experiencia.
