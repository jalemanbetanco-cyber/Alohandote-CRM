# V112 - Barra corporativa de módulos

## Objetivo
Mejorar únicamente la apariencia de la barra principal de módulos sin cambiar lógica ni funcionalidades del sistema.

## Cambios aplicados
- Barra superior con estilo corporativo tipo ERP.
- Agrupación visual:
  - Operación: Renta Car, Alojamientos, Comercial.
  - ERP: Administración, Inventario, RRHH.
  - Control: Mantenimiento, Rentabilidad.
- Íconos discretos por módulo.
- Botón activo más elegante con fondo oscuro, sombra e indicador visual.
- Botones secundarios con fondo blanco cálido, borde suave y hover.
- Contenedor con efecto glass/blur, borde y sombra ligera.
- Responsive mobile manteniendo scroll horizontal.

## No se modificó
- Lógica de reservas.
- Lógica de alojamientos.
- Comercial.
- Administración.
- Inventario.
- RRHH.
- Mantenimiento.
- Rentabilidad.
- Formularios.
- Firebase.
- iCal.
